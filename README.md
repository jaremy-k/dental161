# DentalCare (ДенталКеа) — сайт клиники

Маркетинговый лендинг на Next.js 15 для сети стоматологий [dentalcare161.ru](https://dentalcare161.ru).

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Бренд

- Акцентный цвет: `#00AEEF`
- Логотипы: `public/logo.png`, `public/logo-white.png`

## Сборка

```bash
npm run build
npm start
```
